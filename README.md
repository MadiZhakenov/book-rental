# Book Rental Platform

## Setup

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Environment Variables**
    Copy `.env.example` to `.env` in `apps/web` and `apps/api` (files to be created).

3.  **Start Infrastructure**
    ```bash
    docker-compose up -d
    ```

4.  **Run Development**
    ```bash
    npm run dev
    ```
